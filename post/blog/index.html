使用 Hugo 搭建个人博客
@ 马昊 | Sunday, Dec 20, 2020 | 1 minutes read | Update at Sunday, Dec 20, 2020

关键词 ：Hugo 下载；Git 部署；

关于博客搭建及主题选择的一些步骤笔记~

Step-1 配置 Git 环境

  1. 下载 Git
  2. 配置全局账户及邮箱
git config --global user.name "you Account"
git config --global user.email "you email@163.com"
  1. 生成 SSH 公钥
ssh-keygen -t rsa -C "you email@163.com"

代码参数含义：
    -t 指定密钥类型，默认是 rsa ，可省略
    -C 设置注释文字，邮箱，便于后续分辨文件
    -f 指定密钥文件存储文件名，此处省略
  1. 在 Github 上传 SSH Keys
  2. 测试连接
ssh git@github.com

Step-2 下载并且配置 Hugo

  1. 下载 Hugo
  2. 安装并加入path (win10 环境下)
    • 将 *./bin/Hugo.exe 加入Path
  3. 新建 Hugo 博客项目，选择喜欢的主题
hugo new site yourblog
hugo server -t aHugoTheme --buildDrafts 

其中：
    -t：设置主题
    --buildDrafts：构建草图，在本地预览
  1. 新建 blog
hugo new post/myblog.md
  1. 构建草图（待上传）
hugo --theme=aHugoTheme --baseUrl=http://youRul.io --buildDrafts

Step-3 上传到 Github

  1. 初次提交 Git
git init
git add *
git commit -m"blog内容第一次提交"
  1. 部署远程仓库
git remote add origin https://github.com/yoururl/yoururl.github.io.git
git push -u origin master
